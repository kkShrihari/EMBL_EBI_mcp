# EMBL-EBI MCP Server

A unified **Model Context Protocol (MCP)** server exposing **EMBL-EBI Search**, **Summary (S4)**, and **UniProt Protein** APIs for use in **Claude Desktop** and other MCP-compatible clients.

This project provides **atomic, LLM-friendly tools** for biological search and protein analysis, bundled into a single executable MCP server.

MCP does **not support nested tools**, so each operation is exposed as an **atomic tool**.

Logical grouping is done via prefixes:
- `search_*` â†’ EMBL-EBI Search & Summary
